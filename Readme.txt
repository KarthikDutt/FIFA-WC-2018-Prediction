Steps to Run the code:
1. Run Initial_Predictions.py. This Will create additional csv files in datasets folder.
2. Run Pred_vs_Actual.py. This will compare initial predictions with actual results and capture the unpredictibility 
3. Run Predictions_Final.py. This will generate final predictions keeping in mind the unpredictibility. 

Note - Based on the predictions, obtained the group_16, quarter finalists, semi finalists and finalists are mentioned in the Predictions_Final.py.
They are NOT hardcoded values.
